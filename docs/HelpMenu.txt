## General

  HELP               - Browse command reference  (← → to switch category)
  CL                 - Version change log
  CLEAR              - Clear the active pane
  EXIT               - Close the terminal

## Equity

  Q <Ticker>                       - Live quote
  GIP <Ticker> [Timeframe]         - Price chart
                                     Timeframes: 1W  1M  3M  YTD  1Y  (default 1Y)
                                     ← → to cycle timeframes in pane mode

  DES <Ticker>                     - Company description and profile

  PEERS <Ticker>                   - Peer / competitor companies

  EXEC <Ticker>                    - Executive team and board with compensation

  DIV <Ticker>                     - Dividend history (up to 10 years)

  OWN <Ticker>                     - Insider / ownership transactions
                                     Green = Purchase  Red = Sale

  SENT <Ticker>                    - News sentiment score and buzz metrics
                                     Bull/bear breakdown vs sector average

  FA <Ticker> [Statement] [ANNUAL] - Financial statements
                                     Statements: IS  BS  CF  (default IS)
                                     ← → to switch statement
                                     ↑ ↓  to toggle quarterly / annual

## Macro

  RATES              - U.S. Treasury yield curve with inversion status
                       Overlays today vs 1 month ago

  COMD               - Commodities dashboard
                       Groups: Energy  Metals  Grains

  FX [G10|EM]        - FX major pairs vs USD
                       Screens: G10 (default)  EM
                       ← → to switch screen

  SHIP <Lane>        - Live AIS vessel tracking
                       Lanes: HORMUZ

## Account

  USER               - View your own profile
  USER login         - Sign in  (interactive form — ↑ ↓ navigate, Enter submit)
  USER register      - Create an account  (interactive form)
  USER logout        - Sign out
  USER edit          - Edit bio and location  (interactive form)
  USER <username>    - View another user's profile

## Navigation

  ` (backtick)       - Toggle INPUT mode / PANE mode

  INPUT mode
    Type a command and press Enter to run it.
    ↑ ↓              - Scroll command history
    After Enter, the terminal auto-switches to PANE mode.
    Press ` at any time to return to INPUT mode.

  PANE mode  (enters automatically after every command)
    Z                - Toggle zoom  (focused pane fills screen; others as tabs)
    Tab              - Cycle focused pane
    ↑ ↓              - Scroll or navigate  (command-specific)
    ← →              - Switch tab / timeframe  (GIP, FA, FX, HELP)
